## Description

* Setup DB configuration in .env file 
* Use Postman to test api

## Run Vai DOCKER directly
```bash
$ docker-compose up or docker-compose up -d
```

## Installation

```bash
$ npm install or npm install --force
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
